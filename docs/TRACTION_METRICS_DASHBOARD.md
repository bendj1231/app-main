## Dashboard Overview

The Traction Metrics Dashboard provides investors with a real-time view of PilotRecognition.com's performance, growth trajectory, and key business metrics. This dashboard is designed for investor meetings, board updates, and internal performance tracking.

**Dashboard Sections:**
1. Executive Summary (Top-level KPIs)
2. Revenue Metrics (MRR, ARR, revenue breakdown)
3. User Metrics (Pilot acquisition, engagement, retention)
4. Conversion Metrics (Funnel analysis, conversion rates)
5. Partnership Metrics (Flight schools, airlines, enterprise)
6. Growth Velocity (Month-over-month, year-over-year)
7. Cohort Analysis (Retention, LTV)
8. Projections vs Actuals (Performance against targets)

## Dashboard Implementation Notes

### Data Sources
- **Pilot Data:** Supabase `profiles` table
- **Subscription Data:** Stripe API
- **Referral Data:** Supabase `referrals` table
- **Enterprise Data:** Supabase `enterprise_accounts` table
- **Engagement Data:** Google Analytics + Supabase analytics

### Refresh Frequency
- **Real-time:** Current MRR, active users
- **Daily:** New registrations, conversions
- **Weekly:** Engagement metrics, retention
- **Monthly:** Cohort analysis, LTV calculations

### Access Controls
- **Investor View:** Aggregated metrics only
- **Internal View:** Detailed metrics with drill-down
- **Admin View:** Full access to raw data
